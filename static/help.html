Available Endpoints

Create a user

POST

/user

Call this API Endpoint to create a user object and save it in the SQLite Database.

Parameters:

  • username The username which you want to use
  • password This password will allow you to access the API (hashed using bcrypt)

Create a user

POST

/login

Upon providing correct credentials, a user session will be created using flask-login.

Parameters:

  • username Your account's username
  • password Your account's password

Generate Image

POST

/text2img

Call this API Endpoint to generate an image from a given prompt.

Parameters:

  • prompt The prompt which will be used for the AI Image Generation
  • model See /models, to see all available AI Models.

Query an LLM

POST

/text

Provide a prompt and AI will get you the answer (like ChatGPT).

Parameters:

  • prompt The prompt which will be used for the AI Text Generation
